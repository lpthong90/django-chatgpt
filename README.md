# Features
- Register new user accounts
- Authentication users
- Chat with ChatGPT

# Related Services
- ChatGPT OpenAI => To get `OPENAI_API_KEY`

# Environment Variables
```
OPENAI_API_KEY=<OPENAI_API_KEY>
```


# Run
``` bash
> ./scripts/dev.sh 
```
